# Generated outputs

Use this directory for plots, fabrication files, rendered schematics, and other
exports made from the KiCad source files.

Suggested subdirectories:

- `gerbers/` - Gerber copper, solder mask, silkscreen, and fabrication layers
- `drill/` - Excellon drill files
- `plots/` - PDFs, SVGs, and review renders

The generated subdirectories are ignored by Git; keep only intentionally reviewed
release artifacts under version control.
